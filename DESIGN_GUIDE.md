# 🎨 Class-On 디자인 가이드

> 전자책/동영상 강의 판매 플랫폼의 UI/UX 디자인 컨셉 및 구현 방향

---

## 📌 디자인 철학

**"강사가 5분 만에 자신만의 프리미엄 스토어를 오픈할 수 있도록"**

- **미니멀하면서도 전문적인** - 깔끔하고 신뢰감 있는 디자인
- **직관적인 UX** - 기술 지식 없이도 쉽게 사용
- **모바일 퍼스트** - 모든 화면에서 완벽한 경험
- **커스터마이징 가능** - 강사의 브랜드 색상 적용

---

## 🎯 롤모델 분석

### 1. 해외 플랫폼

#### **Teachable** (teachable.com)
- **강점**: 깔끔한 대시보드, 드래그앤드롭 빌더
- **벤치마크**: 상품 관리 UI, 매출 통계 대시보드
- **참고 요소**:
  - 사이드바 네비게이션 구조
  - 상품 카드 레이아웃
  - 결제 페이지 흐름

#### **Kajabi** (kajabi.com)
- **강점**: 7개 코스 템플릿, 유연한 커스터마이징
- **벤치마크**: 스토어 테마 시스템
- **참고 요소**:
  - 테마 선택기 UI
  - 실시간 미리보기 기능
  - 이메일 마케팅 통합

#### **Thinkific** (thinkific.com)
- **강점**: 다양한 템플릿, 커스텀 CSS 지원
- **벤치마크**: 드래그앤드롭 코스 빌더
- **참고 요소**:
  - 무료 플랜 제공 전략
  - 커뮤니티 기능 UI

## 🎨 디자인 시스템

### 컬러 팔레트

#### **Primary (메인 색상)**
```
Brand Blue: #2563eb (파란색 - 신뢰감, 전문성)
- 주요 CTA 버튼
- 링크
- 강조 요소
```

#### **Secondary (보조 색상)**
```
Accent Purple: #8b5cf6 (보라색 - 창의성)
- 프리미엄 배지
- 할인 태그
- 아이콘
```

#### **Neutral (중립 색상)**
```
Gray 50:  #f9fafb (배경)
Gray 100: #f3f4f6 (카드 배경)
Gray 200: #e5e7eb (테두리)
Gray 500: #6b7280 (보조 텍스트)
Gray 900: #111827 (메인 텍스트)
```

#### **Semantic (의미 색상)**
```
Success: #10b981 (녹색 - 성공, 완료)
Warning: #f59e0b (주황색 - 경고)
Error:   #ef4444 (빨간색 - 에러)
Info:    #3b82f6 (파란색 - 정보)
```

#### **강사 커스터마이징 색상**
- 강사가 Primary 색상을 자유롭게 변경 가능
- 자동으로 hover, active 상태 생성
- WCAG AA 접근성 기준 준수 (대비율 4.5:1 이상)

### 타이포그래피

#### **폰트 패밀리**
```css
/* 한글 */
font-family: 'Pretendard Variable', -apple-system, BlinkMacSystemFont, sans-serif;

/* 영문 (제목) */
font-family: 'Inter', sans-serif;

/* 코드 (개발 강의용) */
font-family: 'JetBrains Mono', 'Fira Code', monospace;
```

#### **폰트 크기 (Tailwind 기준)**
```
Heading 1: text-5xl (48px) - 랜딩 히어로
Heading 2: text-4xl (36px) - 섹션 제목
Heading 3: text-2xl (24px) - 카드 제목
Body:      text-base (16px) - 본문
Small:     text-sm (14px) - 캡션, 보조
Tiny:      text-xs (12px) - 배지, 태그
```

#### **폰트 굵기**
```
Light:    300 (보조 텍스트)
Regular:  400 (본문)
Medium:   500 (강조 텍스트)
Semibold: 600 (제목)
Bold:     700 (주요 헤딩)
```

### 간격 (Spacing)

```
xs:  4px   (0.25rem) - 아이콘 마진
sm:  8px   (0.5rem)  - 버튼 패딩
md:  16px  (1rem)    - 카드 패딩
lg:  24px  (1.5rem)  - 섹션 간격
xl:  32px  (2rem)    - 큰 섹션 간격
2xl: 48px  (3rem)    - 페이지 상단 여백
```

### 보더 & 그림자

#### **Border Radius**
```css
rounded-sm:   2px  - 입력 필드
rounded:      4px  - 버튼
rounded-md:   6px  - 카드
rounded-lg:   8px  - 모달
rounded-xl:   12px - 이미지
rounded-2xl:  16px - 히어로 섹션
rounded-full: 9999px - 아바타, 배지
```

#### **Box Shadow**
```css
/* 카드 */
shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05)
shadow:    0 1px 3px 0 rgb(0 0 0 / 0.1)
shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1)

/* 부유 요소 */
shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1)
shadow-xl: 0 20px 25px -5px rgb(0 0 0 / 0.1)

/* 드롭다운 */
shadow-2xl: 0 25px 50px -12px rgb(0 0 0 / 0.25)
```
